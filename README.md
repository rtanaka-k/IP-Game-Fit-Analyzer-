# 🎮 IP × ゲームジャンル レコメンダー

IPの特性をAIが自動分析し、最適なゲームジャンルと企画コンセプトの「種」を提案するツールです。

## セットアップ

### 必要なもの
- Python 3.9+
- Anthropic API Key

### インストール
```bash
pip install streamlit
```

### 起動
```bash
export ANTHROPIC_API_KEY="your-api-key-here"
streamlit run app.py
```

またはアプリのサイドバーでAPIキーを入力することも可能です。

## 使い方

1. **IP名を入力** — 分析したいIPの名前を入力するだけ
2. **分析結果を確認・修正** — AIが自動生成した属性を確認し、必要に応じて修正
3. **ジャンル推薦を実行** — 確定した属性に基づいて、本命・対抗・大穴の3カテゴリーで提案

## 出力カテゴリー

- **本命（Most Likely）**: 成功確率が高い、王道の提案（2案）
- **対抗（Dark Horse）**: 意外だが、IPの隠れた魅力を引き出す提案（1案）
- **大穴（Wild Card）**: 常識を覆す挑戦的な提案（1案）

## ジャンルデータベース

以下のジャンルが事前登録されています:
- アクションRPG / ターン制RPG
- ストラテジー・タクティクス
- アドベンチャー・ノベルゲーム
- アクション・格闘
- オープンワールド・探索
- リズム・音楽ゲーム
- パズル・カジュアル
- シミュレーション・経営
- サバイバル・クラフト
- 非対称対戦・マルチプレイ
- 位置情報ゲーム

## カスタマイズ

`GENRE_DATABASE` を編集することで、ジャンルの追加・変更が可能です。

## 注意

本ツールの出力は「企画の種」であり、そのまま企画書として使用するものではありません。
発想の起点としてご活用ください。
